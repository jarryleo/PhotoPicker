# PhotoPicker
一款图片选择器，支持单选、多选、裁剪、适配7.0、适配小米

Add it in your root build.gradle at the end of repositories:
```
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```  
Step 2. Add the dependency
```
	dependencies {
	        compile 'com.github.jarryleo:PhotoPicker:v2.0.2'
	}
```
